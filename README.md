# StreetTree Project Page

Project page for **StreetTree: A Large-Scale Global Benchmark for Fine-Grained Tree Species Classification**.

## Live Page

- https://goodgap.github.io/StreetTree_web/

## Edit Content

- Main page content: `index.html`
- Styles: `static/css/index.css`
- JS (carousel, sliders, helpers): `static/js/index.js`
- Images: `static/images/`
- PDFs: `static/pdfs/`
- Videos: `static/videos/`

## Local Preview

GitHub Pages will serve the site as static files. For local preview, run a simple static server in the repo root:

```bash
python -m http.server 8000
```

Then open:

- http://localhost:8000/

## Notes

- Favicon is currently **not set** in `index.html`. If you want one, add a `<link rel="icon">` tag in the `<head>`.
